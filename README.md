# mimi_zeittracking

Zeiterfassungs-App für schnelles Tracking von Arbeitszeiten über mehrere Auftraggeber und Projekte. Automatische Berechnung + flexible Reports mit Export.

## Features (geplant)

- **Quick-Timer** mit Start/Pause/Stop + manuelle Eingabe
- **Auftraggeber → Projekte → Aufgabenbereiche** Hierarchie
- **Automatische Berechnung** von Arbeitszeiten und Überstunden
- **Reports**: Tages-, Monats-, Auftraggeber-, Projekt-Reports
- **PDF/CSV-Export** für Abrechnungen
- **Papierkorb** mit Wiederherstellung

## Tech-Stack

Next.js 14+ | TypeScript | SQLite (Prisma) | Tailwind CSS | shadcn/ui
